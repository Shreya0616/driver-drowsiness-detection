# driver-drowsiness-detection

# Overview
This project detects driver drowsiness in real time using a camera and alerts the driver to prevent accidents.

# Problem Statement
Driver fatigue causes accidents.This system detects drowsiness instantly and triggers alerts.

# Features
Real-time driver monitoring.
Eye and face detection using YOLOv8.
Automatic alert system using buzzer.
Fast and efficient processing.

# Tech Stack
Python
OpenCV
YOLOv8
NumPy

# Hardware
Camera
Laptop/PC
Buzzer

# Working
Capture live video.
Detect face and eyes.
Analyze eye closure.
Trigger alert if drowsy.
